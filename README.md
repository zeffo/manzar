# manzar

cat that wants to catch your mouse. <br>

You can view a live demo on [my website](https://zeffo.me).

## Usage

1. Clone this repo.
1. Install wasm-pack: `cargo install wasm-pack`
1. Build: `wasm-pack build --target web`
